peroscope — View Paper

Peroscope, ISIR CNRS, 2024
Ignacio AVELLINO, Alexandre HADDAD

Peroscope (for intra-operative endoscopy ) is a web interface build on top of the experimental research platform Webstrates and Videostrates to provide a realtime collaborating video editing environment for surgeons. It's provide a additive method by defining and modifying sequence with temporal Tag boxes in a full timeline that appear then in a summary timeline that order and join those sequences.

Main developer to rebuild the interface in order to make it usable into operative context. Adding all the features needed for the study of Alexandre with the supervision of Ignacio AVELLINO.

CPRE — View Paper

CPRE, Sorbonne Université SUMMIT, 2022


Jérome Szewczyk, Aymeric Becq

Project for an automatic guidewire detection solution on fluoroscopic Endoscopic Retrograde Cholangio-Pancreatography (ERCP) images. The aim is to implement automatic guidewire localization in the biliary tract and help reduce the failure rate of this delicate procedure.

Creation of a 3DSlicer module including a 2D/3D fusion algorithm in Python for a diagnostic aid study. This calculates the probability of the guidewire's presence in each bile duct using the 3D model of the patient projected on the fluoroscopy.
